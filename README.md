##  Project Overview

**GadgetHub** is a full-featured e-commerce web application developed using Java Full Stack technologies.  
It follows the MVC architecture and includes both user and admin functionalities.

This project demonstrates strong knowledge of:
- Backend development (JDBC, Servlets)
- Frontend integration (JSP, HTML, CSS)
- Database management (MySQL)
- MVC architecture implementation

---

##  Key Features

### User Module
- User Registration & Login
- Browse Products by Category
- Add to Cart
- Place Orders
- View Order Details
- User Profile Management

### Admin Module
- Admin Login
- Add / Update / Remove Products
- Manage Stock
- View Orders
- Ship / Unship Items

3. Update DB credentials in configuration file
4. Deploy project on Apache Tomcat

##  How to Run

1. Clone the repository:

git clone https://github.com/ashanpratapsing/GadgetHub.git
---------------------------
2. Open in IDE (NetBeans / Eclipse)
3. Configure MySQL
4. Deploy on Tomcat Server
5. Run on:

-------------------------

http://localhost:8080/GadgetHub
--------------------------
